 Nützliche Links

HTML, CSS und JavaScript

  • Mozilla Developer Network: Die beste Dokumentation rund um HTML, CSS und JavaScript, die es gibt. Teilweise auf deutsch, überwiegend auf englisch.
  • SelfHTML: Der Urgroßvater: Gute Sammlung von Erklärungen rund um HTML, CSS, JavaScript. Komplett auf deutsch, aber nicht immer vollständig.
  • W3Schools: Sehr umfangreiche Sammlung von Tutorials zu HTML, CSS und JavaScript. Als zusätzliche Nachschlagequelle gut zu gebrauchen.
  • Can I Use?: Umfangreiche Kompatibilitätstabellen zu allen Browsern. Sehr umfangreich und sehr aktuell.
  • The modern JavaScript tutorial: Sehr umfangreiches JavaScript-Tutorial mit vielen sehr guten Erklärungen.
  • Modern JavaScript Cheatsheet: Übersichtsseite mit allen wesentlichen Neuerungen modernen JavaScripts.

Fortgeschrittene Themen für JavaScript

  • Beginners Guide To Fetching Data With (AJAX, Fetch API & Async/Await): Blogbeitrag, die AJAX-Aufrufe mit XMLHttpRequest und der neueren Fetch API erklärt
  • Google Firebase: Cloudbasierte Datenbank und Laufzeitumgebung für Cloud-Anwendungen
  • HTML5 Local Storage Tutorial: Anleitung zur Entwicklung einer TODO-Liste, deren Inhalte clientseitig gespeichert werden
  • HTML5 Canvas Tutorials: Umfangreiche Anleitungen zur Grafikprogrammierung mit der Canvas API
  • window.requestAnimationFrame(): MDN-Artikel zu window.requestAnimationFrame() für flüssige Animationen mit JavaScript
  • Making your first Phaser 3 game: Framework zur Entwicklung von Browserspielen

Webentwicklung mit Java

  • Java Documentation: API-Dokumentation aller aktuellen Java-Versionen
  • Java EE7 Klassenbibliothek: API-Referenz zur Java Enterprise Edition 7
  • Tutorialspoint: Servlets: Umfangreiche Sammlung von Tutorials rund um Java Servlets.
  • Tutorialspoint: Java Server Pages: Umfangreiche Tutorials rund um Java Server Pages.
